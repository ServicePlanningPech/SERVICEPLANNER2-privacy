SERVICEPLANNER2 Privacy Policy

Last Updated: February 09, 2025

Introduction

This privacy policy explains how SERVICEPLANNER2 ("we", "our", or "the application") collects, uses, and protects your information when you use our church service planning application. We are committed to ensuring that your privacy is protected and to being transparent about how we handle your data.

Information We Collect

Google Account Information

  • Email address: Used for user identification and sending service plan notifications
  • Google Drive access: Required for creating and managing service planning documents

Application Data

  • Service plan details and content
  • Uploaded files and presentations
  • Song selections and arrangements
  • Service notes and comments
  • Additional email addresses (when provided for service plan distribution)

How We Use Your Information

Google Drive Access

SERVICEPLANNER2 requires access to your Google Drive to:

  • Create and manage a "SERVICE PLANS" folder
  • Create and edit service plan documents and presentations
  • Store and manage uploaded files
  • Access template files and song database
  • Create and manage published service plans

Email Usage

We use email functionality to:

  • Send service plans to designated recipients
  • Distribute service information to team members
  • Send notifications about published plans
  • Communicate essential service-related updates

Data Storage

  • All service plans and related files are stored in your Google Drive
  • A settings file (ServicePlanner2Settings) is created to store application preferences
  • Debug logs may be created for troubleshooting purposes

Data Sharing and Privacy

User Control

  • No files or folders are shared without explicit user action
  • Users maintain full control over their service plan data
  • Users can delete service plans and related content at any time

File Sharing

  • Service plan sharing is only initiated through user action
  • Published plans are shared according to user-configured settings
  • Email distribution lists are managed by users

Data Security

Google Services Integration

  • All data is stored securely within Google's infrastructure

Application Security

  • The application runs with limited OAuth scopes
  • All file operations are executed as the authenticated user

User Rights

You have the right to:

  • Access your service plan data at any time
  • Modify or delete any content you've created
  • Control sharing settings for all documents
  • Request debug logs of application activities

Data Retention

  • Service plans remain in your Google Drive until manually deleted
  • Settings are maintained until the application is uninstalled

Technical Details

Google API Services Used

  • Google Drive API
  • Google Slides API
  • Google Drive Activity API
  • Google Apps Script services

Required Permissions

The application requires permissions to:

  • View and manage Google Drive files and folders
  • Create and edit Google Slides presentations
  • Send emails on your behalf
  • Make external API requests (for additional features)

Changes to Privacy Policy

We reserve the right to update this privacy policy at any time. Any changes will be reflected in the "Last Updated" date at the top of this policy. Continued use of PLANNER2 after any modifications to this policy constitutes acceptance of those changes.

Contact Information

For any questions or concerns about this privacy policy or PLANNER2's data handling practices, please contact your system administrator or the application maintainer.

Additional Notes

  • SERVICEPLANNER2 does not collect or store any personal information beyond what is necessary for its core functionality
  • The application does not share any data with third parties
  • All data processing occurs within Google's secure infrastructure
